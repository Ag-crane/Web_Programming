api 요청주소 복사
sample 자리에 인증키
xml 자리에 json

vs code로 볼때는 우클릭 -> 문서서식


1. 빈 데이터는 0 대신 -를 출력
2. 도심권 / 서북권 / 동북권 / 동남권 / 서남권 분류해서 묶어서 border 그리기
3. IDEX_NM 보통=초록색 좋음=파란색 나쁨=빨간색 으로 표시. 글자를.
4. 표에 IDEX_MVL (통합대기환경지수) 추가하고, 상위 다섯 구 파란색, 하위 다섯 구 빨간색으로 표시. 숫자를.


1. 0으로 표기된 데이터는 - 로 바꿔서 표현
2. 권역구 (도심권 ,서남권, ... ) 데이터 사이에 border를 추가하여 구분
3. 각 구별 '보통', '좋음', '나쁨'을 각각 초록, 파랑, 빨간 색으로 표시
4. 통합대기지수 IDEX_MVL 값 상위 5개(숫자가 낮음)은 파란색 bold체로 표시. 하위 5개 구는 빨간색 bold 체로 표시

이외에도 자기가 사는 구를 강조하거나, 표 여백을 조정하여 예쁘게 꾸며주세요
